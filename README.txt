
Requirements
- QT 5.3 win32 -msvc2012
- make sure that QWT is set-up and configured on your QT
- Myo SDK


Instruction:
- change the Myo library Path in the project file to the correct location in your computer
- In the QT Creator, go to Projects and build with the correct kit
  *this current version has been tested with the "Desktop QT 5.3 MSVC2012" kit
- Clean the project
- Build/Rebuild
- Run
